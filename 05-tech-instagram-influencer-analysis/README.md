# 📊 Tech Instagram Influencer Analysis

## 🧾 Project Overview
This project focuses on analyzing the performance of a Tech Instagram Influencer account to identify content trends, audience behavior, and engagement patterns.

The objective is to analyze Instagram activity data using SQL and generate actionable insights to improve reach, engagement, and follower growth.

---

## ❗ Problem Statement
The influencer aims to grow their audience and improve content performance but faces several challenges:

- Identifying high-performing content formats  
- Understanding which categories drive engagement  
- Analyzing follower growth trends  
- Improving reach and visibility  

The goal is to:

- Analyze Instagram data using SQL  
- Identify performance gaps  
- Provide data-driven insights  
- Improve content strategy  

---

## 🎯 Objective

- Analyze Instagram performance using SQL queries  
- Identify top-performing content formats  
- Evaluate engagement metrics (reach, likes, shares)  
- Understand monthly growth trends  
- Provide actionable recommendations  

---

## 📌 Key Metrics

The analysis focuses on the following KPIs:

- Reach Percentage by Content Type  
- Impressions (Highest & Lowest)  
- Total Likes by Category  
- Profile Visits  
- New Followers  
- Total Shares  

---

## 📊 Key Insights

### 🔹 Content Format Strategy
Reels contribute ~62% of total reach, making them the most effective content format for visibility.

### 🔹 Viral Potential of Reels
Reels achieve the highest impressions (up to 339K), indicating strong viral potential.

### 🔹 Best Performing Content Categories
“Other Gadgets” and “Tech Tips” generate the highest engagement in terms of likes.

### 🔹 Growth Breakthrough Month
May recorded the highest profile visits and follower growth, indicating peak performance.

### 🔹 Share Behavior Insight
Image posts receive the highest number of shares, making them effective for awareness.

---

## 🚀 Recommendations

- Increase focus on Reel-based content  
- Create more gadget-related and educational posts  
- Replicate strategies used during high-growth months  
- Use high-quality images for shareability  
- Experiment with content variations to improve engagement
