# 📊 SQL Outputs – Analysis Results

## 🧾 Overview
This folder contains the output results generated from SQL queries used in the Tech Instagram Influencer Analysis project.

The outputs represent structured insights derived from the dataset, helping to understand content performance, audience engagement, and growth patterns.

---

## 🎯 Objective
- Present results of SQL-based analysis
- Validate insights derived from query execution
- Support business decision-making with data-driven outputs

---

## 📁 Files Included
- Output files corresponding to each SQL query (`output_01` to `output_10`)
- Each output represents the result of a specific analytical question

---

## 🔍 Key Insights Derived

### 1. Post Type Analysis
- Identified different types of posts (Image, Reel, Video, Carousel)
- Compared performance metrics across post types

### 2. Impressions & Reach Trends
- Evaluated highest and lowest impressions by post type
- Identified which formats generate maximum visibility

### 3. Engagement Metrics
- Analyzed likes, comments, shares, and saves
- Determined which categories drive higher engagement

### 4. Monthly Growth Analysis
- Tracked profile visits and new followers across months
- Identified peak growth periods

### 5. Category Performance
- Compared performance across product categories (Mobile, Laptop, Earphone, etc.)
- Highlighted top-performing categories based on engagement

### 6. Content Strategy Insights
- Identified high-performing content formats (e.g., Reels dominating reach)
- Provided direction for optimizing future content strategy

---

## 🛠️ Tools & Technologies
- SQL (MySQL)
- Query Execution & Result Analysis

---

## 💡 Key Takeaways
- Reels contribute the highest reach percentage among all post types
- Certain categories consistently outperform others in engagement
- Monthly trends indicate clear growth patterns and peak periods
- Data-driven insights can significantly improve content strategy decisions

## 📌 Note
These outputs are directly generated from SQL queries and are used to support the insights and recommendations presented in the main project dashboard and report.
