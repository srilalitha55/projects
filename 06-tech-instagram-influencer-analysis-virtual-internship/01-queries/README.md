# 📊 SQL Queries – Data Analysis

## 🧾 Overview
This folder contains SQL queries used to analyze the Tech Instagram Influencer dataset as part of the virtual internship project.

The queries are designed to extract meaningful insights related to content performance, audience engagement, and growth trends.

---

## 🎯 Objective
- Analyze influencer data using SQL
- Identify high-performing content and engagement patterns
- Support data-driven decision-making through structured queries

---

## 📁 Files Included
- `query_01.sql` to `query_10.sql`
- Each query focuses on a specific business question or analytical requirement

---

## 🔍 Key Analysis Areas

### 1. Content Performance
- Identify top-performing posts based on engagement metrics
- Analyze reach and interaction levels across content types

### 2. Audience Engagement
- Evaluate likes, comments, and overall engagement trends
- Compare engagement across different posts and categories

### 3. Growth Trends
- Track follower growth and performance over time
- Identify patterns in content that drive audience growth

### 4. Category & Format Insights
- Analyze which content categories perform best
- Understand the impact of different formats (videos, posts, etc.)

---

## 🛠️ Tools & Technologies
- SQL (MySQL)

---

## 💡 Key Takeaways
- SQL was used to clean, filter, and analyze structured data
- Queries helped uncover actionable insights for content strategy
- Enabled identification of high-performing content patterns
